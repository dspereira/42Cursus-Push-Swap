# 42Cursus-Push-Swap

## Installation / Usage

This project is designed to work on Linux and macOS.

Clone repo:
```shell
git clone git@github.com:dspereira/42Cursus-Push-Swap.git push_swap && cd push_swap
```

Build:
```shell
make
```

Usage example:
```shell
./push_swap [list of numbers]
```
